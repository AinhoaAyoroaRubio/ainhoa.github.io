# DOA · Página de presentación para Foro de Empleo

Página web de presentación del proyecto **DOA**, desarrollada para el **Foro de Empleo de la UPV Gandia**.

El objetivo de esta página es ofrecer una presentación rápida, clara y profesional del proyecto, mostrando el propósito de la aplicación, el equipo de desarrollo, las tecnologías utilizadas y las vías de contacto profesional.

---

## Descripción del proyecto

**DOA** es una plataforma web educativa orientada a facilitar la conexión entre docentes y alumnos.

El proyecto plantea una solución digital para organizar tareas, entregas, perfiles de usuario y flujos básicos dentro de un entorno académico, diferenciando las necesidades de cada tipo de usuario.

La página incluida en este repositorio funciona como una landing page de presentación del equipo y del MVP desarrollado.

---

## Objetivo de la página

Esta web ha sido creada para:

- Presentar el proyecto DOA en un contexto profesional.
- Mostrar de forma clara quiénes forman parte del equipo.
- Facilitar el acceso a perfiles profesionales y contacto.
- Servir como carta de presentación rápida durante el Foro de Empleo.
- Permitir que empresas y visitantes accedan fácilmente mediante URL o código QR.

## Cómo ejecutar el proyecto en local

No requiere instalación ni dependencias externas.

Para visualizar la página en local:

1. Descarga o clona el repositorio.
2. Abre el archivo `index.html` en un navegador.
3. También puede ejecutarse mediante una extensión como **Live Server** en Visual Studio Code.

---

## Publicación con GitHub Pages

Para publicar la página:

1. Subir los archivos a la rama `main`.
2. Entrar en la configuración del repositorio.
3. Ir a `Settings > Pages`.
4. Seleccionar:
   - Branch: `main`
   - Folder: `/root`
5. Guardar los cambios.
6. Esperar a que GitHub Pages genere la publicación.

La URL final del proyecto es:

```txt
https://ainhoaayoroarubio.github.io/ainhoa.github.io/
```
