## Cómo publicarlo en GitHub Pages

1. Entra en el repositorio `ainhoayoroarubio.github.io`.
2. Sube estos archivos en la raíz del repositorio:
   - `index.html`
   - `styles.css`
3. Haz commit en la rama `main`.
4. Entra a `https://ainhoaayoroarubio.github.io/ainhoa.github.io/`.
