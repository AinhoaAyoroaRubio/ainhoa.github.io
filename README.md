# DOA · Página de presentación para Foro de Empleo

Página web de presentación del proyecto **DOA**, desarrollada para el **Foro de Empleo de la UPV Gandia**.

El objetivo de esta página es ofrecer una presentación rápida, clara y profesional del proyecto, mostrando el propósito de la aplicación, el equipo de desarrollo, las tecnologías utilizadas y los enlaces de contacto profesional de sus integrantes.

---

## Descripción del proyecto

**DOA** es una plataforma web educativa orientada a facilitar la conexión entre docentes y alumnos.

El proyecto plantea una solución digital para organizar tareas, entregas, perfiles y flujos básicos dentro de un entorno académico, diferenciando las necesidades de cada tipo de usuario.

La página incluida en este repositorio funciona como una landing page de presentación del equipo y del MVP desarrollado.

---

## Objetivo de la página

Esta web ha sido creada para:

- Presentar el proyecto DOA en un contexto profesional.
- Mostrar de forma clara quiénes forman parte del equipo.
- Facilitar el acceso a LinkedIn o vías de contacto.
- Servir como carta de presentación rápida durante el Foro de Empleo.
- Tener una página pública accesible mediante GitHub Pages.
